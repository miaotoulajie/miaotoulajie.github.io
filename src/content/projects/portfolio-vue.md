这个项目的核心目标是把一个初始 demo 网站，升级成可以长期运营的个人品牌站。

我把页面结构重构为首页、项目、文章、关于和联系，并补齐了基础 SEO 能力。

在数据侧，我将 Supabase 接入分层到 infrastructure、repository、service，确保后续可以平滑迁移到独立 API 服务。
